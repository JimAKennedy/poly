Poly's audio previews use a hand-picked drum library sourced from openly-licensed sample repositories. The CC-BY entries below carry the TASL attribution string their license requires — every playback surface on this site is tied to one of these sources through `site/public/samples/manifest.json`, so this page cannot drift out of sync with what actually plays.
